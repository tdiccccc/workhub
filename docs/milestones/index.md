# Milestones

## 概要

WorkHubは完成形を一度に実装するのではなく、MVPから段階的に機能追加を行います。

各MilestoneはGitHub Milestoneと対応しており、Issue、Pull Request、学習内容を管理する単位となります。

## Definition of Done

- 全Issue完了
- PRレビュー完了
- 動作確認済み
- ドキュメント更新済み
- 学習ログ作成済み

---

## 運用方針

- GitHub Milestoneと本ドキュメントのMilestoneを1対1で対応させる
- Issueは必ずいずれかのMilestoneに紐づける
- Milestone完了時に振り返りを記録する
- 完成形を一度に実装せず、リリース可能な単位で継続的に開発する
