# Milestone v0.5

## ゴール

検索、ソート、ページネーションを追加し、一覧画面を実務で使いやすい状態にする。

---

## 機能

- Project検索
- Task検索
- WorkLog検索
- ソート
- ページネーション
- 日付範囲フィルタ
- ステータスフィルタ

---

## 学習内容

- Spring Data JPA Pageable
- Sort
- Specification
- 動的検索条件
- クエリパラメータ設計
- Nuxt4でのURLクエリ管理
- 一覧UI設計

---

## 完了条件

- 一覧APIでページネーションできる
- 主要一覧画面で検索とソートができる
- URLクエリと画面状態が同期している
- 検索条件を変更しても画面が破綻しない
- API仕様書に検索仕様が反映されている

---

## レビュー項目

- クエリパラメータ名が統一されているか
- ページングのレスポンス形式が統一されているか
- 検索条件が増えてもServiceが肥大化しすぎていないか
- 一覧画面の操作性が良いか
